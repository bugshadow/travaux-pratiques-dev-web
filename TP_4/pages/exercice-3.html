3.1 Barre de navigation
HTML 
<nav class="navbar">
  <div class="logo">MonSite</div>
  <ul class="nav-links">
    <li><a href="#">Accueil</a></li>
    <li><a href="#">Services</a></li>
    <li><a href="#">Contact</a></li>
    </ul>
</nav>
2
Réponse — Barre de navigation avec Flexbox
style.css 
/* Barre de navigation */
.navbar {
  display: flex;
  justify-content: space-between; /* logo à gauche, liens à droite */
  align-items: center; /* centrage vertical */
  background-color: #333;
  padding: 15px 30px;
}

.logo {
  color: white;
  font-weight: bold;
  font-size: 20px;
}

.nav-links {
  display: flex;
  list-style: none;
  gap: 20px; /* espace entre les liens */
}

.nav-links a {
  color: white;
  text-decoration: none;
}

.nav-links a:hover {
  color: #0066cc;
}

3.2 Cartes de même hauteur
HTML 
<div class="cards">
  <div class="card">
    <h3>Carte 1</h3>
    <p>Contenu court.</p>
    <a href="#" class="card-link">En savoir plus</a>
  </div>
  /* ... autre cartes ... */
</div>
5
Réponse — Cartes avec Flexbox et footer aligné
style.css 
/* Conteneur de cartes */
.cards {
  display: flex;
  gap: 20px; /* espace entre les cartes */
  margin: 20px 0;
}

.card {
  flex: 1; /* largeur égale */
  display: flex;
  flex-direction: column; /* contenu en colonne */
  background-color: white;
  border: 1px solid #ddd;
  padding: 20px;
  border-radius: 5px;
}

.card h3 {
  margin-bottom: 10px;
}

.card p {
  flex-grow: 1; /* pousse le contenu vers le haut */
  margin-bottom: 15px;
}

.card-link {
  background-color: #0066cc;
  color: white;
  padding: 10px 15px;
  text-decoration: none;
  border-radius: 3px;
  align-self: flex-start; /* bouton en bas */
}

/* Responsive */
@media (max-width: 700px) {
  .cards {
    flex-direction: column; /* cartes empilées */
  }
}

Concepts clé — À retenir !
flex: 1

Distribue l'espace restant entre les items. flex: 1 = largeur égale pour tous les enfants.

flex-grow: 1

Étire l'item pour occuper l'espace vertical disponible. Utile pour pousser le bouton au bas de la carte.

justify-content: space-between

Logo à gauche, liens à droite avec espace maximum entre eux. Parfait pour les barres de navigation.
