Exercice 3 : Flexbox

Alignement et distribution des éléments

3.1 Barre de navigation
Code HTML (Donné)
<nav class="navbar">
  <div class="logo">MonSite</div>
  <ul class="nav-links">
    <li><a href="#">Accueil</a></li>
    <li><a href="#">Services</a></li>
    <li><a href="#">Contact</a></li>
  </ul>
</nav>
Solution CSS
.navbar {
  display: flex;
  justify-content: space-between;
  align-items: center;
}
.nav-links {
  display: flex;
  gap: 20px;
  list-style: none;
}
3.2 Cartes de même hauteur
Solution CSS (Alignement vertical)
.cards {
  display: flex;
  gap: 20px;
}

.card {
  flex: 1;
  display: flex;
  flex-direction: column;
}

.card p {
  flex-grow: 1; /* Pousse le lien vers le bas */
}
