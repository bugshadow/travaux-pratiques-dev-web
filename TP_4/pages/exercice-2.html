Box Model CSS

Comprendre la largeur · Padding · Margin · Border · box-sizing

1
2.1 Comprendre le Box Model
HTML 
/* Élément avec box model */
.box {
  width: 300px;
  padding: 20px;
  border: 5px solid #333;
  margin: 15px;
}
2
Réponses — 2.1 Box Model
Q1 : content-box (défaut)
Largeur totale visible ?
width: 300px
+ padding: 20px × 2 = 40px
+ border: 5px × 2 = 10px
= 350px
(margin 15px n'est pas inclus, c'est l'espace externe)
Q2 : border-box
Largeur totale visible ?
= 300px
Tout est inclus dans les 300px : width + padding + border
border-box inclut padding et border dans la largeur définie
3
Réponse — Q3 : Reset CSS universel
style.css 
/* Reset universel pour box-sizing */
* {
  box-sizing: border-box;
}

/* Alternative : seulement tous les éléments et pseudo-éléments */
*, *::before, *::after {
  box-sizing: border-box;
}
4
Réponses — 2.2 Centrer un élément
Méthode 1 : Flexbox (RECOMMANDÉ) 
/* Conteneur parent */
.container {
  display: flex;
  justify-content: center; /* horizontal */
  align-items: center; /* vertical */
  height: 100vh; /* hauteur plein écran */
}

/* Enfant */
.box {
  width: 600px;
}
Méthode 2 : Grid 
.container {
  display: grid;
  place-items: center;
  height: 100vh;
}

.box {
  width: 600px;
}
Méthode 3 : Horizontal uniquement 
.box {
  width: 600px;
  margin: 0 auto;
}
Méthode 4 : Positionnement absolu 
.container {
  position: relative;
  height: 100vh;
}

.box {
  position: absolute;
  top: 50%;
  left: 50%;
  transform: translate(-50%, -50%);
  width: 600px;
}

7
Concepts clés à retenir
▸
content-box (défaut)
Width = contenu uniquement. Padding et border s'ajoutent à la largeur totale. Comportement surprenant pour les débutants.
≈
border-box (recommandé)
Width inclut contenu + padding + border. Beaucoup plus intuitif et prévisible. À utiliser partout !
∿
Centrage flexible
Flexbox et Grid rendent le centrage simple. Préférez-les à positionnement absolu ou margin auto.
