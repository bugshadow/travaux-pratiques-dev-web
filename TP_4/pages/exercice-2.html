Exercice 2 : Box Model

Padding, Border, Margin et Centrage

2.1 Comprendre le Box Model
CSS : Box Definition
.box {
  width: 300px;
  padding: 20px;
  border: 5px solid #333;
  margin: 15px;
}
Questions & Réponses
  1. Largeur totale avec content-box (défaut) :
    300px (width) + 2*20px (padding) + 2*5px (border) = 350px
  2. Largeur totale avec border-box :
    La largeur totale visible de l'élément est égale à la valeur définie : 300px.
  3. Reset CSS universel pour box-sizing :
    * { box-sizing: border-box; }
2.2 Centrer un élément
Centrage complet (Horizontal & Vertical)
/* Centrage horizontal seul (margin auto) */
div {
  width: 600px;
  margin-left: auto;
  margin-right: auto;
}

/* Centrage complet avec Flexbox */
body {
  display: flex;
  justify-content: center;
  align-items: center;
  min-height: 100vh;
}
