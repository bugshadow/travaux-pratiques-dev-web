7.1 Bouton avec Transitions
CSS (Transitions) 
.btn-anim {
  background-color: #0066cc;
  transition: all 0.3s ease;
}

.btn-anim:hover {
  background-color: #004499;
  transform: scale(1.1);
  box-shadow: 0 5px 15px rgba(0,0,0,0.3);
}

7.2 Animation de chargement (@keyframes)
CSS (Keyframes) 
@keyframes spin {
  0% { transform: rotate(0deg); }
  100% { transform: rotate(360deg); }
}

.spinner {
  border: 4px solid #f3f3f3;
  border-top: 4px solid #0066cc;
  border-radius: 50%;
  width: 40px;
  height: 40px;
  animation: spin 1s linear infinite;
}

Concepts clé — À retenir !
transition

Permet de modifier les valeurs des propriétés de manière fluide sur une durée donnée. Syntaxe : propriété durée timing-function.

transform

Modifie la forme et la position d'un élément (scale, rotate, translate) sans impacter le layout des autres éléments.

@keyframes

Définit les étapes d'une animation. On l'applique avec la propriété animation.
