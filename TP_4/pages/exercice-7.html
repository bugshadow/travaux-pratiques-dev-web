Exercice 7 : Transitions & Animations

Interactivité et mouvement en CSS

7.1 Bouton animé (Transitions)
Solution : Transition au survol
.btn-anim {
  padding: 12px 24px;
  background-color: #333;
  color: white;
  border: none;
  cursor: pointer;
  transition: transform 0.3s ease, background-color 0.3s ease;
}

.btn-anim:hover {
  background-color: #bb5544;
  transform: scale(1.1);
}
7.2 Animation de chargement (Spinner)
Solution : Spinner @keyframes
@keyframes rotate-spinner {
  0% { transform: rotate(0deg); }
  100% { transform: rotate(360deg); }
}

.spinner {
  width: 40px;
  height: 40px;
  border: 4px solid #eee;
  border-top-color: #bb5544;
  border-radius: 50%;
  animation: rotate-spinner 1s linear infinite;
}
