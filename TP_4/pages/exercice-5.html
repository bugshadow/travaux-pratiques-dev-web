Exercice 5 : Responsive Design

Mobile First et Media Queries

5.1 Mobile First (Tablette & Desktop)
CSS : Base Mobile First
/* Base pour mobile (déjà donnée) */
.container {
  width: 100%;
  padding: 15px;
}

.nav-menu {
  display: flex;
  flex-direction: column;
  gap: 5px;
}

.content { display: block; }
.sidebar { display: none; }
Solution : Media Queries (Tablette & Desktop)
/* Tablette (min-width: 768px) */
@media (min-width: 768px) {
  .nav-menu {
    flex-direction: row;
    justify-content: space-between;
  }
}

/* Desktop (min-width: 1024px) */
@media (min-width: 1024px) {
  .container {
    display: flex;
  }
  .sidebar {
    display: block;
    width: 250px;
  }
  .content { flex: 1; }
}
