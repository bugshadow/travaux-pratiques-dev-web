5.1 Approche Mobile First
CSS (Code de base) 
/* Base mobile (Mobile First) */
.container {
  width: 100%;
  padding: 15px;
}

.nav-menu {
  display: flex;
  flex-direction: column;
  gap: 10px;
}

.sidebar {
  display: none; /* Caché sur mobile */
}
2
Réponse — Ajout des Media Queries
style.css 
/* Tablette (min-width: 768px) */
@media (min-width: 768px) {
  .nav-menu {
    flex-direction: row; /* Menu horizontal */
  }
  .container {
    max-width: 750px;
    margin: 0 auto;
  }
}

/* Desktop (min-width: 1024px) */
@media (min-width: 1024px) {
  .container {
    max-width: 1000px;
    display: grid;
    grid-template-columns: 250px 1fr;
  }
  .sidebar {
    display: block; /* Affiché sur desktop */
  }
}

4
Concepts clé — À retenir !
Mobile First

On définit d'abord le style pour mobile, puis on ajoute les contraintes pour les écrans plus grands avec min-width.

Media Queries

Utilisez `@media` pour appliquer des styles uniquement si une condition (ex: largeur d'écran) est remplie.

Breakpoints

Les points de rupture communs sont souvent 768px (tablettes) et 1024px (ordinateurs).
