Exercice 4 : CSS Grid Layout

Grilles responsive et Layout avec Areas

4.1 Grille de cartes responsive
CSS : grid-template-columns
/* Grille avec nombre de colonnes automatique */
.grid-cards {
  display: grid;
  grid-template-columns: repeat(auto-fit, minmax(200px, 1fr));
  gap: 20px;
}
4.2 Layout complet avec Grid Areas
CSS : Mise en page complète
/* Définition des zones de grille */
.page-layout {
  display: grid;
  grid-template-areas: 
    "header header"
    "sidebar main"
    "footer footer";
  grid-template-columns: 250px 1fr;
  grid-template-rows: auto 1fr auto;
  min-height: 100vh;
}

/* Attribution des éléments aux zones */
.header { grid-area: header; }
.sidebar { grid-area: sidebar; }
.main { grid-area: main; }
.footer { grid-area: footer; }
