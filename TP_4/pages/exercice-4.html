4.1 Grille de cartes responsive
HTML 
<div class="grid-cards">
  <div class="card">Carte 1</div>
  <div class="card">Carte 2</div>
  <div class="card">Carte 3</div>
  <div class="card">Carte 4</div>
  <div class="card">Carte 5</div>
  <div class="card">Carte 6</div>
</div>
2
Réponse — Grille avec CSS Grid
style.css 
/* Conteneur de grille */
.grid-cards {
  display: grid;
  grid-template-columns: repeat(auto-fit, minmax(200px, 1fr));
  gap: 20px; /* espace entre les cartes */
  margin: 20px 0;
}

.grid-cards .card {
  background-color: white;
  border: 1px solid #ddd;
  padding: 20px;
  border-radius: 5px;
  text-align: center;
  font-weight: bold;
}

4.2 Layout complet avec Grid Areas
HTML 
<div class="page-layout">
  <header class="header">Header</header>
  <nav class="sidebar">Sidebar</nav>
  <main class="main">Contenu principal</main>
  <footer class="footer">Footer</footer>
</div>
5
Réponse — Layout avec grid-template-areas
style.css 
/* Conteneur principal */
.page-layout {
  display: grid;
  grid-template-areas: 
    "header header"
    "sidebar main"
    "footer footer";
  grid-template-columns: 250px 1fr; /* sidebar fixe, main fluide */
  grid-template-rows: auto 1fr auto;
  gap: 15px;
  min-height: 400px;
}

.page-layout .header { grid-area: header; }
.page-layout .sidebar { grid-area: sidebar; }
.page-layout .main { grid-area: main; }
.page-layout .footer { grid-area: footer; }

/* Éléments internes pour présentation */
.page-layout > * {
  background-color: #eef2f5;
  border: 1px dashed #99aab5;
  padding: 20px;
  display: flex;
  align-items: center;
  justify-content: center;
  font-weight: bold;
  color: #333;
}

/* Responsive mobile */
@media (max-width: 768px) {
  .page-layout {
    grid-template-areas: 
      "header"
      "sidebar"
      "main"
      "footer";
    grid-template-columns: 1fr;
  }
}

Concepts clé — À retenir !
repeat(auto-fit, minmax(...))

Crée des colonnes responsives sans media queries. auto-fit remplit l'espace avec autant de colonnes que possible de largeur minimum (ex: 200px).

grid-template-areas

Permet de "dessiner" visuellement la mise en page en nommant les zones à l'aide de chaînes de caractères. Très intuitif !

grid-area

Associe un élément HTML à un nom de zone définit dans la grille parente, permettant de le placer facilement.
