6.1 Système de Variables CSS
CSS (Définition) 
/* Thème clair (Défaut) */
:root {
  --primary-color: #0066cc;
  --bg-color: #ffffff;
  --text-color: #333333;
  --card-bg: #f9f9f9;
}

/* Thème sombre */
[data-theme="dark"] {
  --primary-color: #4da3ff;
  --bg-color: #1a1a1a;
  --text-color: #f0f0f0;
  --card-bg: #2d2d2d;
}
2
Réponse — Utilisation des Variables
style.css 
body {
  background-color: var(--bg-color);
  color: var(--text-color);
  transition: background 0.3s, color 0.3s;
}

.btn-primary {
  background-color: var(--primary-color);
  color: white;
  padding: 10px 20px;
  border-radius: 5px;
}

4
Concepts clé — À retenir !
--var-name

Les variables CSS commencent toujours par deux tirets (--). Elles sont sensibles à la casse.

var(--name)

On accède à la valeur d'une variable via la fonction var(). On peut ajouter une valeur de secours : var(--color, blue).

:root

Déclarer les variables dans :root les rend accessibles dans tout le document (portée globale).
