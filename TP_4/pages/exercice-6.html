Exercice 6 : Variables CSS & Thèmes

Propriétés personnalisées et maintenance

6.1 Système de couleurs (Clair / Sombre)
Solution : Définition des variables (:root)
/* Thème Clair (Défaut) */
:root {
  --bg-color: #ffffff;
  --text-color: #333333;
  --primary: #bb5544;
}

/* Thème Sombre */
[data-theme="dark"] {
  --bg-color: #1a1a1a;
  --text-color: #f0f0f0;
  --primary: #ff7766;
}
Utilisation des variables en CSS
body {
  background-color: var(--bg-color);
  color: var(--text-color);
}

.btn {
  background-color: var(--primary);
}
